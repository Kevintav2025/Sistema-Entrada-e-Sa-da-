# 🔄 Sistema de Controle de Entrada e Saída

Este é um sistema web desenvolvido para o gerenciamento e monitoramento de fluxos de entrada e saída. A aplicação foi estruturada para oferecer uma interface clara e objetiva para o registro de movimentações, consultas de histórico e geração de relatórios de controle.

## 💻 Telas Integradas

O sistema conta com módulos focados na facilidade de operação e registro de dados:

* **Página Inicial (`index.html`):** Painel principal com resumo das últimas movimentações e atalhos rápidos para registro.
* **Autenticação (`login.html`):** Tela para identificação e login dos operadores do sistema.
* **Registro de Entradas:** Interface dedicada para dar entrada de novos itens, valores ou registros no sistema.
* **Registro de Saídas:** Interface configurada para a baixa, retirada ou registros de saída.
* **Histórico / Consultas:** Listagem completa e filtros para busca de movimentações passadas.

*(Nota: Se os arquivos HTML do seu repositório tiverem nomes diferentes destes, basta editar esta lista diretamente no arquivo).*

---

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação dos formulários de cadastro e tabelas de exibição de dados.
* **CSS3:** Design limpo, responsivo e focado na usabilidade do operador.
* **C# / JavaScript:** Lógica para persistência de dados, validação de campos e controle de fluxo.
